## 開発環境
- Python
- Choregraphe（総合開発環境）

## 特徴 - Features
Pepperの受付アプリケーション。声での応答やタッチパネルでの応対、自動移動などを実装しています。

### 1. 声での応対
会場内の展示物の場所やトイレの位置などを聞けば応答してくれます。

### 2. タッチパネルでの応対
普段触り慣れているタッチパネル操作で簡単にイベント内の情報が入手できます。

### 3. 自動移動
人を認識してPepperを触って貰えるよう近づきます。

## 使用方法
1.  **Choregraphe を起動しプロジェクトを起動します。**
2.  **Pepperに接続**
3.  **アップロードして再生をクリック**
